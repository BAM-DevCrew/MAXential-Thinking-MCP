## PR Description
### What is this?
MAXential Thinking MCP is an enhanced reasoning server that provides 11 focused tools for structured thinking, branching, revision, and thought navigation. It's a v2 redesign of the reference sequential-thinking server with improved architecture.

### Key Features
- **11 specialized tools** for different thinking operations
- **Branching support** for exploring alternative thought paths
- **Revision tracking** for refining previous thoughts
- **Session management** for complex multi-step reasoning
- **TypeScript implementation** with clean architecture
